# Lab 19: JavaScript Background Color Timer

## Experiment
Write a JavaScript program to change background color after 5 seconds of page load.

## Objective
Learn to use JavaScript timing functions (`setTimeout` and `setInterval`) to create timed events and dynamic visual changes on a web page.

## Instructions
1. Open `index.html` and `script.js` in VS Code
2. Use the color values and content from `content.md`
3. Use `setTimeout()` to change the page background color after 5 seconds
4. Add a visible countdown display showing "Background will change in X seconds..."
5. **Bonus:** Use `setInterval()` to cycle through multiple colors every 3 seconds
6. Add buttons to Start, Stop, and Reset the color cycle
7. Test by loading the page and waiting 5 seconds
8. Save and preview in Chrome

## Expected Output
The page background changes color after 5 seconds. Optionally, it cycles through multiple colors with start/stop controls.

## Grading Criteria
- [ ] Background color changes after 5 seconds using `setTimeout()` (3 marks)
- [ ] `setTimeout()` used correctly (3 marks)
- [ ] Color cycle through multiple colors using `setInterval()` (3 marks)
- [ ] Countdown/timer display visible on page (3 marks)
- [ ] Clean, well-commented code (3 marks)

**Total: 15 marks**
