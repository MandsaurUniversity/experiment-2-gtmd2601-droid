# Lab 18: JavaScript Alert on Page Load

## Experiment
Write a JavaScript program to display an information box as soon as page loads.

## Objective
Learn to use JavaScript dialog boxes (`alert`, `confirm`, `prompt`) and the `window.onload` event to interact with users when a page loads.

## Instructions
1. Open `index.html` and `script.js` in VS Code
2. Use the content and dialog messages from `content.md`
3. In `script.js`, use `window.onload` to display a welcome alert when the page loads
4. Create a university welcome page with the sections described in `content.md`
5. Add buttons that demonstrate:
   - `alert()` — displays an information message
   - `confirm()` — asks a yes/no question and handles the response
   - `prompt()` — asks for user input and displays a personalized greeting
6. Test by refreshing the page (alert should appear immediately)
7. Save and preview in Chrome

## Expected Output
A welcome alert appears on page load, and the page has buttons demonstrating all three JavaScript dialog types.

## Grading Criteria
- [ ] Alert displays on page load using `window.onload` (3 marks)
- [ ] Meaningful welcome message in the alert (3 marks)
- [ ] Page content loads properly after dismissing alert (3 marks)
- [ ] Additional dialogs demonstrated — `confirm()` and `prompt()` (3 marks)
- [ ] Clean, well-commented code (3 marks)

**Total: 15 marks**
