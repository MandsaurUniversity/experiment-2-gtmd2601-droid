# Lab 07: Div and Span Layout

## Experiment
Use `<span>` and `<div>` tags to provide a layout to the above page instead of a table layout.

## Objective
Learn to use `<div>` and `<span>` tags along with CSS to create page layouts. You will recreate the university infrastructure page from Lab 06, but this time using `<div>` for block-level layout and `<span>` for inline styling instead of tables.

## Instructions
1. Open `index.html` in VS Code
2. Use the same infrastructure content from Lab 06 (referenced in `content.md`)
3. Use `<div>` tags with `id` attributes for major page sections: header, sidebar, main-content, footer
4. Use `<span>` tags for inline text styling (highlighting keywords, colouring text)
5. Write CSS in `style.css` to control layout using `float`, `width`, `padding`, `margin`
6. Link the CSS file using `<link rel="stylesheet" href="style.css">`
7. Create infrastructure cards using `<div>` elements with CSS styling
8. Save and preview in Chrome

## Expected Output
A visually appealing page with the same infrastructure content as Lab 06, but using div/span-based layout with CSS instead of HTML tables.

## Grading Criteria
- [ ] Proper use of `<div>` for block-level page layout (3 marks)
- [ ] Proper use of `<span>` for inline text styling (3 marks)
- [ ] CSS file linked and used for layout (float/width/padding) (3 marks)
- [ ] Content matches Lab 06 infrastructure items (3 marks)
- [ ] Clean, readable, well-indented HTML and CSS code (3 marks)

**Total: 15 marks**
