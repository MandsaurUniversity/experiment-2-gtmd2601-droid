# Lab 06: University Infrastructure Table

## Experiment
Use tables to provide layout to your HTML page describing your university infrastructure.

## Objective
Learn to use HTML tables not just for tabular data but also as a layout tool to arrange content on a webpage. You will create a page showcasing Mandsaur University's infrastructure using a table-based layout with images and descriptions.

> **Note:** Table-based layouts were common in early web design. Modern websites use CSS for layout (which you'll learn in Lab 07). However, understanding table layouts is part of the syllabus.

## Instructions
1. Open `index.html` in VS Code
2. Use the content from `content.md` — descriptions of 8 university infrastructure items
3. Create a table-based layout with 2 columns: image on the left, description on the right
4. Each infrastructure item should be a table row
5. Include at least 6 infrastructure items
6. Use placeholder images (URLs provided in content.md) or add your own in the `images/` folder
7. Add proper headings, borders, and padding
8. Save and preview in Chrome

## Expected Output
A well-organized page showcasing university infrastructure using a table-based layout, with images alongside descriptions.

## Grading Criteria
- [ ] Table-based page layout with proper structure (3 marks)
- [ ] At least 6 infrastructure items displayed (3 marks)
- [ ] Images with descriptions for each item (3 marks)
- [ ] Proper table attributes (border, cellpadding, width) (3 marks)
- [ ] Clean, readable, well-indented HTML code (3 marks)

**Total: 15 marks**
