# Lab 04: Background Colors and Anchor Navigation

## Experiment
Change the background color of the page. At the bottom create a link to take the user to the top of the page.

## Objective
Learn to apply background colors to an HTML page using the `bgcolor` attribute or inline CSS, and create internal anchor links for page navigation (jumping to the top of the page).

## Instructions
1. Open `index.html` in VS Code
2. Use the content from `content.md` — it's a long article about Mandsaur city
3. Set a background color on the `<body>` tag (suggested: `#f0f8ff`, `#fffff0`, or `#f5f5dc`)
4. Add an anchor at the top of the page: `<a id="top"></a>` or `<h1 id="top">...</h1>`
5. Add multiple sections of content so the page is long enough to scroll
6. At the bottom of the page, add a "Back to Top" link: `<a href="#top">↑ Back to Top</a>`
7. Save and preview in Chrome — scroll down and test the "Back to Top" link

## Expected Output
A coloured webpage with a long article about Mandsaur city. Clicking the "Back to Top" link at the bottom scrolls the page to the top.

## Grading Criteria
- [ ] Background color applied to the page (3 marks)
- [ ] Sufficient content to require scrolling (3 marks)
- [ ] Anchor defined at the top of the page with `id` attribute (3 marks)
- [ ] "Back to Top" link at the bottom that works correctly (3 marks)
- [ ] Clean, readable, well-indented HTML code (3 marks)

**Total: 15 marks**
