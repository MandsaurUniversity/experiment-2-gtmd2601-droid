# Lab 30: Bootstrap Form Validation

## Experiment
Utilize Bootstrap's built-in validation styles and JavaScript functionality to validate a form's input fields.

## Objective
Learn to use Bootstrap's built-in form validation classes (`was-validated`, `is-valid`, `is-invalid`, `valid-feedback`, `invalid-feedback`) along with JavaScript to create a professional form validation experience.

## Instructions
1. Open `index.html` and `script.js` in VS Code
2. Use the form specification from `content.md`
3. Create a student registration form using Bootstrap form classes
4. Implement validation using:
   - Bootstrap's `needs-validation` class on the form
   - `required` attribute on mandatory fields
   - Custom JavaScript validation for complex rules
   - Bootstrap's `valid-feedback` and `invalid-feedback` divs
5. The form should show green borders for valid fields and red borders for invalid fields
6. Test with both valid and invalid data

## Grading Criteria
- [ ] Bootstrap form with proper classes (form-control, form-label) (3 marks)
- [ ] Bootstrap validation styles (was-validated, valid/invalid feedback) (3 marks)
- [ ] All required fields validated (3 marks)
- [ ] Custom JavaScript validation for complex rules (3 marks)
- [ ] Clean, professional appearance (3 marks)

**Total: 15 marks**
