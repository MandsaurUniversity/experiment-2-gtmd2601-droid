# Lab 14: Bootstrap Shopping Website

## Experiment
Design a web page in Bootstrap for a shopping website.

## Objective
Learn to use Bootstrap 5 components (cards, grid, navbar, buttons) to create a professional-looking e-commerce product listing page.

## Instructions
1. Open `index.html` in VS Code
2. Use the product data from `content.md`
3. Create a shopping page with:
   - A navbar with the store name and navigation links
   - A hero/jumbotron section with a welcome banner
   - Product cards arranged in a responsive grid (3 columns on desktop, 2 on tablet, 1 on mobile)
   - Each card should have: product image, name, price in ₹, description, and "Add to Cart" button
   - A footer with store information
4. Use Bootstrap classes — do NOT write custom CSS unless necessary
5. Save and preview in Chrome

## Expected Output
A responsive shopping page that looks professional on both desktop and mobile screens.

## Grading Criteria
- [ ] Bootstrap CDN linked correctly, responsive navbar (3 marks)
- [ ] Product cards with images, prices, descriptions (3 marks)
- [ ] Responsive grid layout (col-lg-4, col-md-6, col-12) (3 marks)
- [ ] Proper use of Bootstrap components (buttons, badges, etc.) (3 marks)
- [ ] Clean code and visual appeal (3 marks)

**Total: 15 marks**
