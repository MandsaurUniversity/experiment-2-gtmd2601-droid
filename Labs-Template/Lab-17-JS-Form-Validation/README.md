# Lab 17: JavaScript Form Validation

## Experiment
Create a form in HTML and put validation checks on values entered by the user using JavaScript (such as age should be a value between 1 and 150).

## Objective
Learn to validate form inputs using JavaScript before submission, ensuring data integrity and user-friendly error messages.

## Instructions
1. Open `index.html` and `script.js` in VS Code
2. Use the form fields and validation rules from `content.md`
3. Create an HTML form with all specified fields
4. Write JavaScript validation in `script.js` that:
   - Validates each field when the form is submitted
   - Shows appropriate error messages for invalid inputs
   - Prevents form submission if any field is invalid
   - Uses `alert()` or inline error messages to notify the user
5. Test with both valid and invalid data
6. Save and preview in Chrome

## Expected Output
A form that validates all inputs before submission and shows clear error messages for invalid data.

## Grading Criteria
- [ ] All form fields present with correct input types (3 marks)
- [ ] JavaScript validation for all fields (3 marks)
- [ ] Appropriate error messages displayed (3 marks)
- [ ] Form prevents submission on invalid data (3 marks)
- [ ] Clean, well-commented code (3 marks)

**Total: 15 marks**
