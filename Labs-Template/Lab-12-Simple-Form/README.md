# Lab 12: Simple Form to Submit User Input

## Experiment
Create a simple form to submit user input like his name, age, address and favorite subject, movie and singer.

## Objective
Learn to create HTML forms using various input types (`text`, `number`, `textarea`) and dropdown menus (`<select>`) to collect user information.

## Instructions
1. Open `index.html` in VS Code
2. Refer to `content.md` for the form field specifications and sample data
3. Create an HTML form with the following fields:
   - **Full Name** — text input with placeholder
   - **Age** — number input with min/max constraints
   - **Address** — textarea with multiple rows
   - **Favorite Subject** — dropdown `<select>` with BCA subjects
   - **Favorite Movie** — dropdown `<select>` with Indian movies
   - **Favorite Singer** — dropdown `<select>` with Indian singers
4. Every input field must have a corresponding `<label>` with a `for` attribute
5. Use appropriate `name` attributes for all fields
6. Add a Submit button at the end
7. Save and preview in Chrome

## Expected Output
A clean, functional HTML form that collects a student's name, age, address, and preferences using text fields, a textarea, and dropdown menus.

## Grading Criteria
- [ ] Proper `<form>` structure with method and action attributes (3 marks)
- [ ] Correct input types — text, number, textarea (3 marks)
- [ ] Dropdown `<select>` elements with `<option>` for favorites (3 marks)
- [ ] `<label>` elements linked to every input field (3 marks)
- [ ] Clean, well-indented HTML code (3 marks)

**Total: 15 marks**
