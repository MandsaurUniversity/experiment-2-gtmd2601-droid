# Lab 03: Image Links

## Experiment
Insert an image and create a link such that clicking on the image takes the user to another page.

## Objective
Learn to insert images using the `<img>` tag and make images clickable by wrapping them inside anchor (`<a>`) tags. You will create a multi-page website where clicking on images navigates to a second page.

## Instructions
1. Open `index.html` in VS Code
2. Use the content provided in `content.md` as your source material
3. Insert at least 3 images using `<img>` tags with proper `alt` text
4. Wrap each image inside an `<a>` tag so clicking the image navigates to `page2.html`
5. Edit `page2.html` to contain information about Mandsaur University
6. Add a "Back to Home" link on `page2.html` pointing back to `index.html`
7. If you have your own images, place them in the `images/` folder
8. Save and preview in Chrome — test that clicking images navigates correctly

## Expected Output
A homepage with 3 clickable images. Clicking any image takes the user to a second page (`page2.html`) with university information and a link back to the homepage.

## Grading Criteria
- [ ] Images inserted with proper `alt` text and dimensions (3 marks)
- [ ] Images wrapped in `<a>` tags — clickable and navigating correctly (3 marks)
- [ ] Second page (`page2.html`) created with meaningful content (3 marks)
- [ ] Navigation works both ways — home to page2 and back (3 marks)
- [ ] Clean, readable, well-indented HTML code (3 marks)

**Total: 15 marks**
