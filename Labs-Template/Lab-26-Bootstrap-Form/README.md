# Lab 26: Bootstrap Responsive Form

## Experiment
Design a responsive form using Bootstrap with fields for name, email, and message, along with a submit button.

## Objective
Learn to create a professional, responsive contact form using Bootstrap's form classes and grid system.

## Instructions
1. Open `index.html` in VS Code
2. Use the form specification from `content.md`
3. Create a responsive contact form with:
   - A form title and description
   - Fields for Full Name, Email, Phone, Subject (dropdown), and Message (textarea)
   - Proper Bootstrap form classes (`form-control`, `form-label`, `mb-3`, `form-select`)
   - A two-column layout: form on left (`col-md-8`), contact info on right (`col-md-4`)
   - A styled submit button using Bootstrap classes
4. Add proper `required` attributes on mandatory fields
5. Test responsive behavior — form should stack on mobile

## Grading Criteria
- [ ] Bootstrap form classes used correctly (form-control, form-label, mb-3) (3 marks)
- [ ] All required fields present (name, email, phone, subject, message) (3 marks)
- [ ] Responsive two-column layout (form + contact info) (3 marks)
- [ ] Submit button styled with Bootstrap (btn, btn-primary) (3 marks)
- [ ] Clean code and proper structure (3 marks)

**Total: 15 marks**
