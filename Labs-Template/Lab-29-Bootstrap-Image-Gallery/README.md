# Lab 29: Bootstrap Responsive Image Gallery

## Experiment
Create a responsive grid layout using Bootstrap to display a gallery of images, ensuring they adjust appropriately based on screen size.

## Objective
Learn to create a responsive image gallery using Bootstrap's grid system, image utilities, and card components.

## Instructions
1. Open `index.html` in VS Code
2. Use the gallery content from `content.md`
3. Create an image gallery with:
   - At least **12 images** arranged in a responsive grid
   - Each image in a Bootstrap **card** or rounded container with a caption
   - Responsive breakpoints:
     - Large screens (`lg`): 4 images per row → `col-lg-3`
     - Medium screens (`md`): 3 images per row → `col-md-4`
     - Small screens (`sm`): 2 images per row → `col-sm-6`
     - Extra small: 1 image per row → `col-12`
4. Add a gallery title and description at the top
5. Optional: Add category filter buttons (All, Campus, Labs, Events, Sports)
6. Optional: Add hover effects or a modal preview on image click
7. Test responsive behavior at different screen sizes

## Grading Criteria
- [ ] At least 12 images displayed in the gallery (3 marks)
- [ ] Responsive grid with correct breakpoints (3 marks)
- [ ] Image styling (rounded corners, cards, or similar) (3 marks)
- [ ] Hover effects or modal preview on click (3 marks)
- [ ] Clean code and professional layout (3 marks)

**Total: 15 marks**
