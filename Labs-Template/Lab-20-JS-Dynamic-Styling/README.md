# Lab 20: JavaScript Dynamic Styling

## Experiment
Write a JavaScript program to dynamically bold, italic and underline words and phrases based on user actions.

## Objective
Learn to manipulate CSS styles dynamically using JavaScript DOM properties, allowing users to toggle text formatting through button interactions.

## Instructions
1. Open `index.html` and `script.js` in VS Code
2. Use the paragraphs and button labels from `content.md`
3. Create a page with 3 paragraphs (each with a unique `id`)
4. Add a toolbar with buttons for Bold, Italic, Underline, Reset All, and Change Color
5. In `script.js`, write functions that:
   - Toggle bold on a selected paragraph (`style.fontWeight`)
   - Toggle italic on a selected paragraph (`style.fontStyle`)
   - Toggle underline on a selected paragraph (`style.textDecoration`)
   - Reset all paragraphs to default styling
6. **Bonus:** Add a color picker to change text color and font size buttons
7. Test by clicking buttons and verifying styles toggle on/off
8. Save and preview in Chrome

## Expected Output
A page with paragraphs and a button toolbar. Clicking a style button toggles that style on the selected paragraph.

## Grading Criteria
- [ ] Bold toggle works correctly (3 marks)
- [ ] Italic toggle works correctly (3 marks)
- [ ] Underline toggle works correctly (3 marks)
- [ ] Styling applies to correct elements (3 marks)
- [ ] Clean, well-commented code (3 marks)

**Total: 15 marks**
