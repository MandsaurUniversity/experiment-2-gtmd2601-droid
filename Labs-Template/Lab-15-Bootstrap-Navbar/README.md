# Lab 15: Bootstrap Navigation Bar

## Experiment
Make a navigation bar in Bootstrap.

## Objective
Learn to create a fully responsive Bootstrap 5 navbar with brand logo, navigation links, dropdown menus, and an integrated search form.

## Instructions
1. Open `index.html` in VS Code
2. Use the navigation structure from `content.md`
3. Create a page with:
   - A responsive navbar with brand name "Mandsaur University" and placeholder logo
   - Navigation links: Home, About Us, Academics (dropdown), Admissions, Campus Life, Contact
   - Academics dropdown with programme names: BCA, BBA, B.Com, B.Sc (CS), M.Com, MBA, MCA
   - A search form with placeholder "Search courses..." and a Search button
   - The navbar should collapse into a hamburger menu (toggler) on mobile
   - Page content below the navbar (welcome section, quick links, footer)
4. Use Bootstrap classes — do NOT write custom CSS unless necessary
5. Save and preview in Chrome; test by resizing the browser window

## Expected Output
A responsive navigation bar that collapses on small screens, with working dropdown menus and a search form. The page below should have enough content to demonstrate scroll behaviour.

## Grading Criteria
- [ ] Responsive navbar with toggler button for mobile (3 marks)
- [ ] Dropdown menu for Academics with programme list (3 marks)
- [ ] Search form integrated into the navbar (3 marks)
- [ ] Active link state and proper navigation structure (3 marks)
- [ ] Clean code, semantic HTML, and visual appeal (3 marks)

**Total: 15 marks**
