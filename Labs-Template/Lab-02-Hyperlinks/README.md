# Lab 02: Hyperlinks

## Experiment
Create links on the words e.g., "Wi-Fi" and "LAN" to link them to Wikipedia pages.

## Objective
Learn to use the HTML anchor tag (`<a>`) to create hyperlinks. You will write paragraphs about networking technologies and turn specific technical terms into clickable links pointing to their Wikipedia articles.

## Instructions
1. Open `index.html` in VS Code
2. Use the content provided in `content.md` as your source material
3. Write at least 3 paragraphs about networking and internet technologies
4. Identify at least 8 technical terms in your text (Wi-Fi, LAN, WAN, Bluetooth, Ethernet, TCP/IP, HTTP, DNS)
5. Wrap each term with an `<a>` tag linking to its Wikipedia page
6. Use `target="_blank"` so links open in a new tab
7. Save and preview in Chrome — test that all links work

## Expected Output
An informative webpage with multiple paragraphs where technical terms are clickable hyperlinks leading to their respective Wikipedia pages.

## Grading Criteria
- [ ] Proper use of `<a>` tags with `href` attribute (3 marks)
- [ ] Correct Wikipedia URLs for all terms (3 marks)
- [ ] Meaningful anchor text — terms naturally embedded in paragraphs (3 marks)
- [ ] Proper HTML page structure with headings (3 marks)
- [ ] Clean, readable, well-indented HTML code (3 marks)

**Total: 15 marks**
