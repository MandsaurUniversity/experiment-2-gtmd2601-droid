# Lab 25: Bootstrap Page Layout

## Experiment
Create a simple web page layout using Bootstrap that includes a navigation bar, a header with a large heading, and a section with three columns containing images and captions.

## Objective
Learn to create a complete page layout using Bootstrap's grid system, navbar, and typography components.

## Instructions
1. Open `index.html` in VS Code
2. Use the content from `content.md`
3. Create a page with:
   - A responsive **navbar** with brand name and links
   - A **header section** with a large heading (use Bootstrap's display classes or jumbotron-style)
   - A **three-column section** using Bootstrap grid (each column has an image and a caption)
4. The three columns should be responsive:
   - Desktop: 3 columns side by side (`col-lg-4`)
   - Tablet: 2 columns (`col-md-6`)
   - Mobile: 1 column (`col-12`)
5. Save and preview in Chrome — test responsive behavior

## Grading Criteria
- [ ] Responsive navbar with brand and links (3 marks)
- [ ] Header with large heading (Bootstrap typography) (3 marks)
- [ ] Three-column responsive grid with images and captions (3 marks)
- [ ] Proper Bootstrap classes used throughout (3 marks)
- [ ] Clean code and responsive behavior (3 marks)

**Total: 15 marks**
