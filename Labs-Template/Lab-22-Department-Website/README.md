# Lab 22: Department Website Project

## Experiment
Using ideas from previous experiments, create a complete website for your department.

## Objective
Apply all HTML, CSS, JavaScript, and Bootstrap skills learned so far to build a multi-page, responsive department website.

## Instructions
1. This is a larger project — plan your pages before coding
2. Use the content from `content.md` as your source material
3. Create at least 6 pages (listed below)
4. Use Bootstrap for responsive layout
5. Maintain consistent navigation across all pages
6. Add CSS styling for a professional look
7. Use JavaScript for interactivity (form validation, dynamic content)

## Required Pages
1. **index.html** — Home page with welcome message, highlights, and quick links
2. **about.html** — About the department (history, vision, mission)
3. **courses.html** — Courses offered with details
4. **faculty.html** — Faculty profiles with photos
5. **events.html** — Upcoming and past events
6. **contact.html** — Contact form with address and map

## Grading Criteria
- [ ] All 6 pages created with proper content (3 marks)
- [ ] Consistent Bootstrap navbar across all pages (3 marks)
- [ ] Responsive design (works on mobile) (3 marks)
- [ ] CSS styling and visual appeal (3 marks)
- [ ] JavaScript interactivity (form validation, dynamic content) (3 marks)

**Total: 15 marks**

Combine techniques from all previous labs. See course content in `2-Content/` folders 2 through 7.
