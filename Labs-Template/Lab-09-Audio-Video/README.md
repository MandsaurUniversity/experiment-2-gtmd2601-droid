# Lab 09: Audio and Video

## Experiment
Embed Audio and Video into your HTML web page.

## Objective
Learn to embed multimedia content (audio and video) in an HTML page using the HTML5 `<audio>` and `<video>` elements. You will also explore embedding external content using `<iframe>`.

## Instructions
1. Open `index.html` in VS Code
2. Use the media sources and content from `content.md`
3. Add an `<audio>` element with `controls` attribute and at least one audio source
4. Add a `<video>` element with `controls`, `width`, and `poster` attributes
5. Try the following attributes: `autoplay`, `loop`, `muted`
6. Add an `<iframe>` to embed a YouTube video
7. Add fallback text inside `<audio>` and `<video>` tags for unsupported browsers
8. Write 2-3 paragraphs about multimedia in web design
9. Save and preview in Chrome — test that audio and video play correctly

## Expected Output
A multimedia webpage with an embedded audio player, a video player, and a YouTube video via iframe, along with descriptive paragraphs about multimedia in web design.

## Grading Criteria
- [ ] Audio element with `controls` and working audio source (3 marks)
- [ ] Video element with `controls`, `width`, and working video source (3 marks)
- [ ] Proper use of attributes: `autoplay`, `loop`, `muted`, `poster` (3 marks)
- [ ] Fallback text for unsupported browsers (3 marks)
- [ ] Clean, readable, well-indented HTML code (3 marks)

**Total: 15 marks**
