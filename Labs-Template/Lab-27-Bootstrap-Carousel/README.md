# Lab 27: Bootstrap Carousel/Slider

## Experiment
Develop a carousel/slider using Bootstrap to showcase multiple images with navigation arrows and indicators.

## Objective
Learn to create an interactive image carousel using Bootstrap's Carousel component with controls, indicators, and captions.

## Instructions
1. Open `index.html` in VS Code
2. Use the slide content from `content.md`
3. Create a Bootstrap carousel with:
   - At least **5 slides** with images
   - **Previous/Next navigation arrows** (carousel controls)
   - **Dot indicators** at the bottom to show current slide
   - **Captions** on each slide (title and description text)
4. The carousel should auto-play and allow manual navigation
5. Add content below the carousel (a paragraph about Mandsaur)
6. Test in Chrome — verify arrows, indicators, and auto-slide work

## Grading Criteria
- [ ] At least 5 slides with images (3 marks)
- [ ] Working previous/next navigation arrows (3 marks)
- [ ] Dot indicators showing current slide (3 marks)
- [ ] Captions on each slide (title + description) (3 marks)
- [ ] Clean code and proper Bootstrap structure (3 marks)

**Total: 15 marks**
