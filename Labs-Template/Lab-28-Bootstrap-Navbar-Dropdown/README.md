# Lab 28: Bootstrap Responsive Navbar with Dropdowns

## Experiment
Develop a responsive navbar with dropdown menus using Bootstrap for easy navigation on both desktop and mobile devices.

## Objective
Learn to create a fully responsive navigation bar with multiple dropdown menus using Bootstrap's Navbar and Dropdown components.

## Instructions
1. Open `index.html` in VS Code
2. Use the navigation structure from `content.md`
3. Create a responsive navbar with:
   - A **brand name** (and optional logo)
   - Multiple **regular nav links** (Home, About Us, Placements, Contact Us)
   - At least **2 dropdown menus** (Departments, Admissions, Student Life)
   - A **responsive toggler** (hamburger menu) for mobile devices
4. Below the navbar, add:
   - A hero section with a welcome message
   - Info cards showing university statistics
   - A footer with links
5. Use `navbar-dark bg-dark` and `sticky-top` for the navbar
6. Test on both desktop and mobile — verify toggler and dropdowns work

## Grading Criteria
- [ ] Responsive navbar with working toggler for mobile (3 marks)
- [ ] At least 2 dropdown menus with proper items (3 marks)
- [ ] Dropdown items relevant and well-organized (3 marks)
- [ ] Works correctly on mobile (toggler + dropdowns) (3 marks)
- [ ] Clean code and professional appearance (3 marks)

**Total: 15 marks**
