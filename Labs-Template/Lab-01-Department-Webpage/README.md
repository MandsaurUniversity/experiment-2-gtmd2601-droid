# Lab 01: Department Webpage

## Experiment
Create a webpage with HTML describing your department. Use paragraph and list tags.

## Objective
Learn to create a basic HTML webpage using paragraph (`<p>`) and list (`<ul>`, `<ol>`, `<li>`) tags to describe the Department of Computer Applications at Mandsaur University.

## Instructions
1. Open `index.html` in VS Code
2. Use the content provided in `content.md` as your source material
3. Structure the page with proper headings (`<h1>` to `<h3>`)
4. Write at least 3 paragraphs using `<p>` tags describing the department
5. Create an unordered list (`<ul>`) of courses offered
6. Create an ordered list (`<ol>`) of department achievements
7. Add a list of faculty members
8. Save and preview in Chrome

## Expected Output
A well-structured HTML page about the Department of Computer Applications with multiple paragraphs and both ordered and unordered lists.

## Grading Criteria
- [ ] Proper HTML structure with `<!DOCTYPE>`, `<html>`, `<head>`, `<body>` (3 marks)
- [ ] At least 3 paragraphs using `<p>` tags (3 marks)
- [ ] Unordered list of courses/facilities (3 marks)
- [ ] Ordered list of achievements/rankings (3 marks)
- [ ] Clean, readable, well-indented HTML code (3 marks)

**Total: 15 marks**
