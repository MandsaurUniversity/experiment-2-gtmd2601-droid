# Lab 16: Bootstrap Image Slider (Carousel)

## Experiment
Make an image slider in Bootstrap.

## Objective
Learn to create a Bootstrap 5 carousel (image slider) with navigation arrows, slide indicators, and captions overlaid on each slide.

## Instructions
1. Open `index.html` in VS Code
2. Use the slide data from `content.md`
3. Create a page with:
   - A Bootstrap carousel with at least 5 slides
   - Each slide should have an image, a title, and a descriptive caption
   - Previous and Next navigation arrows
   - Slide indicators (dots/dashes) at the bottom of the carousel
   - Captions positioned over each image using `carousel-caption`
   - A brief "About" section below the carousel
4. Use Bootstrap classes — do NOT write custom CSS unless necessary
5. Save and preview in Chrome; test the arrows and indicators

## Expected Output
A fully functional image slider that auto-advances through slides, with working prev/next controls and clickable indicators.

## Grading Criteria
- [ ] Carousel with 5 or more slides and images (3 marks)
- [ ] Working previous/next navigation controls (3 marks)
- [ ] Slide indicators (dots) that are clickable (3 marks)
- [ ] Captions with title and description on each slide (3 marks)
- [ ] Clean code, semantic HTML, and visual appeal (3 marks)

**Total: 15 marks**
