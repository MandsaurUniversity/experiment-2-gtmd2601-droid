# Lab 05: Class Timetable

## Experiment
Create a table to show your class time-table.

## Objective
Learn to create HTML tables using `<table>`, `<tr>`, `<th>`, and `<td>` tags. Practice using `colspan` for cells spanning multiple columns (breaks/lunch) and `rowspan` for cells spanning multiple rows (lab sessions).

## Instructions
1. Open `index.html` in VS Code
2. Use the timetable data from `content.md`
3. Create a table with a caption: "BCA Semester II — Class Timetable"
4. Use `<th>` for the header row (Day, Time Slots)
5. Use `colspan` for Break and Lunch rows that span all time columns
6. Use `rowspan` for 2-hour lab sessions that span two time slots
7. Add a border to the table using the `border` attribute or inline CSS
8. Save and preview in Chrome

## Expected Output
A well-formatted HTML table showing the BCA Semester II weekly timetable with proper use of colspan for breaks and rowspan for lab sessions.

## Grading Criteria
- [ ] Correct table structure with `<table>`, `<tr>`, `<th>`, `<td>` (3 marks)
- [ ] Proper use of `colspan` for Break and Lunch rows (3 marks)
- [ ] Proper use of `rowspan` for 2-hour lab sessions (3 marks)
- [ ] Table caption and header row with day/time labels (3 marks)
- [ ] Clean, readable, well-indented HTML code (3 marks)

**Total: 15 marks**
