# Lab 10: Inline CSS Styling

## Experiment
Apply inline CSS to change colors of certain text portions, bold, underline and italicize certain words in your HTML web page. Also change the background color of each paragraph using inline CSS.

## Objective
Learn to use the `style` attribute to apply CSS directly to HTML elements for text formatting and background colors.

## Instructions
1. Open `index.html` in VS Code
2. Use the content from `content.md` as your source text
3. Create at least 5 paragraphs, each with a different background color
4. Within each paragraph, apply inline CSS to:
   - Change text color of specific words (use `color` property)
   - Make certain words bold (use `font-weight: bold`)
   - Underline important terms (use `text-decoration: underline`)
   - Italicize descriptions (use `font-style: italic`)
5. Use at least 5 different colors
6. Save and preview in Chrome

## Expected Output
A colorful webpage with distinct paragraph backgrounds and various text styling applied inline.

## Grading Criteria
- [ ] At least 5 paragraphs with different background colors (3 marks)
- [ ] Text color changes on specific words (3 marks)
- [ ] Bold, underline, and italic applied correctly (3 marks)
- [ ] Visually appealing and readable design (3 marks)
- [ ] Clean, well-indented HTML code (3 marks)

**Total: 15 marks**
