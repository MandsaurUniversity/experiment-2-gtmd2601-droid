## Left Frame — Navigation (`nav.html`)

The left frame should contain a list of links. Each link should use `target="content"` so it opens in the center frame.

### Links to Include:
1. **Home** — links to `main.html`
2. **About Department** — links to `about.html` (create this file if you want, or just use `main.html` with different sections)
3. **Courses** — links to `courses.html`
4. **Faculty** — links to `faculty.html`
5. **Contact** — links to `contact.html`

> **Tip:** For simplicity, you can create separate HTML files for each page, OR you can keep all content in `main.html` and use anchor links (`#about`, `#courses`, etc.)

### Styling Suggestions for Nav Frame:
- Background color: `#2c3e50` (dark blue)
- Text/link color: `white`
- Font: Arial or sans-serif
- Padding: 15px

### About Department Page
The Department of Computer Applications was established in 2005 as part of Mandsaur University's commitment to providing world-class technical education. The department is accredited by NAAC and follows a curriculum aligned with industry standards and UGC guidelines.

Our vision is to become a centre of excellence in computer applications education, producing technically competent professionals who contribute to the nation's IT growth. The department has a dedicated team of 8 faculty members, including 2 PhDs and 6 postgraduates with specialisations in Cloud Computing, Data Science, AI/ML, Web Technologies, Information Security, and Software Engineering.

### Courses Page
**Programmes Offered:**
- **BCA (Bachelor of Computer Applications)** — 3-year undergraduate programme covering programming, web development, database management, networking, and software engineering. Annual fee: ₹45,000.
- **MCA (Master of Computer Applications)** — 2-year postgraduate programme with advanced topics in AI, cloud computing, and data science. Annual fee: ₹60,000.
- **PGDCA (Post Graduate Diploma in Computer Applications)** — 1-year diploma for graduates seeking IT skills. Fee: ₹30,000.
- **Certificate in Web Development** — 6-month certificate course covering HTML, CSS, JavaScript, and Bootstrap. Fee: ₹15,000.
- **Certificate in Python Programming** — 6-month course covering Python basics, data structures, and libraries. Fee: ₹15,000.
