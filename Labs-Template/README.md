## How to Use These Templates

### For Instructors

1. **Create a new repository** on GitHub under the `MandsaurUniversity` organization
   - Name it like: `template-lab-01-department-webpage`
2. **Copy the contents** of the corresponding folder (e.g., `Lab-01-Department-Webpage/`) into the new repo
3. Go to the repo **Settings** → Check **"Template repository"**
4. In **GitHub Classroom**, create a new assignment and select this template
5. Share the assignment link on **Google Classroom** (Code: `zxfnnqag`)

### For Students

- You **don't** use these folders directly
- Your instructor creates assignments from these templates
- You accept assignments via links posted on [Google Classroom](https://classroom.google.com/c/ODU4MzYxMzU0MTI5?cjc=zxfnnqag)
